# number types
there are 2 types of numerical data that we can use in pyhton: Integer and float
## 1.  integers
'integer' is a definition we use for whole numbers. such as 5 25 -34 these are some examples.
## 2.   Floats 
Floats are decimal numbers. They can be **positive** or **negative**.

* Example (Positive): `pi = 3.14`
* Example (Negative): `temperature = -10.5`

>   *Note: Python treats `-5.0` as a *float, even if it looks like a whole number. 
>   Because it has a dot!
